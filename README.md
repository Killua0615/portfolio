# Portfolio Website

Personal portfolio site for Natsumi Teshima — Software Engineer & Marketer based in Tokyo.

## Pages

- **Home** (`index.html`) — Introduction with watercolor animation
- **Engineering** (`engineering.html`) — Software engineering projects (work, personal, school)
- **Marketing & Branding** (`marketing.html`) — Marketing and branding experience
- **About** (`about.html`) — Bio, skills, photography, and links

## Features

- EN / JP language toggle (persisted via localStorage)
- Watercolor background animation (canvas)
- Responsive design
- No frameworks — built with vanilla HTML, CSS, and JavaScript

## Tech Stack

- HTML / CSS / JavaScript
- No build tools or dependencies
